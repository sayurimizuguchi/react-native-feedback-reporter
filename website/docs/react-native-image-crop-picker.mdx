## Setup

1. Install [react-native-image-crop-picker](https://github.com/ivpusic/react-native-image-crop-picker)
2. Pass `extraSource` prop with library name to enable extra image upload functionality
```js
import { FeedbackReporter } from 'react-native-feedback-reporter';

<FeedbackReporter
  extraSource="react-native-image-crop-picker"
/>
```
3. The extra image attachment source will show up to the right of the screenshot
