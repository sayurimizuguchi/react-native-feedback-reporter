### Get library

With npm:
```bash
npm install react-native-feedback-reporter
```

With Yarn:
```bash
yarn add react-native-feedback-reporter
```

### Link

#### Android & iOS

- **React Native 0.60+**

[CLI autolink feature](https://github.com/react-native-community/cli/blob/master/docs/autolinking.md) links the module while building the app.

On iOS, use CocoaPods to add the native `FeedbackReporter` to your project:

```bash
npx pod-install
```

- **React Native <= 0.59**


```bash
react-native link react-native-feedback-reporter
```
